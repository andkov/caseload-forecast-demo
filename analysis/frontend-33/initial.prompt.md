## Every frontend is a website

Every frontend is a static Quarto website with a standard architecture:

- **Project** — mission, method, glossary, summary
- **Pipeline** — pipeline guide, data manifest
- **Analysis** — EDA reports, non-exploratory reports
- **Docs** — README, site map, publisher notes

These four sections are mandatory. If you want to publish a talk or presentation, it goes under an optional **Story** section within this website — not as a standalone deliverable.
